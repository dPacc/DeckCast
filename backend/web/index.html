Video Enhancement

Steam Deck records at its native 1280x800 resolution. Enhanced downloads upscale to higher resolutions using FFmpeg with Lanczos filtering and sharpening for a crisper image on larger screens.

Enhancement runs on the Steam Deck's CPU. A 20-minute clip may take 5-10 minutes to process. The result is cached so subsequent downloads are instant.

Setup Guide

  1. Go to console.cloud.google.com
  2. Create a new project (or select an existing one)
  3. Search for and enable the YouTube Data API v3
  4. Navigate to APIs & Services → Credentials
  5. Click Create Credentials → OAuth Client ID
  6. Set application type to Desktop application
  7. Click Create, then Download JSON
  8. Upload the downloaded file above

About DeckCast

Share your Steam Deck game recordings over Wi-Fi. Browse, preview, download, enhance, and organize your clips from any device.

Download — Original quality, direct from the Deck
Enhanced Download — Upscaled with Lanczos + sharpening
Folders — Virtual folders to organize clips
YouTube Upload — Upload directly with OAuth
Preview — Stream and preview clips in-browser
